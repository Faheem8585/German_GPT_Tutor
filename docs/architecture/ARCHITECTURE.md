- **Medium-term (Redis, TTL 90d)**: Mistake patterns, XP, streaks, vocabulary sets
- **Long-term (PostgreSQL)**: Learning sessions, progress, achievements
- **Semantic (Qdrant)**: User memory embeddings for personalized retrieval

## Key Engineering Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Agent framework | LangGraph | Stateful graph execution, production-ready |
| LLM routing | Provider abstraction | Avoid vendor lock-in, cost optimization |
| RAG fusion | RRF over weighted sum | No parameter tuning required, robust |
| Memory | Redis + PG | Fast session + durable history |
| Voice STT | OpenAI Whisper | Best German accuracy |
| Auth | JWT (stateless) | Horizontal scaling friendly |
| Eval | LLM-as-judge | Scalable quality assurance |
| Streaming | SSE | Simpler than WebSocket for one-way |
